# HUESOS — лучший пакет-менеджер на районе ☠️
Лицензия: [HUESOS LICENSE](LICENSE) — делай что хочешь мне поебать
---
### Что умеет:
- `huesos пакет.имя` — ставит проги одной командой  
- `huesos add myrepo https://site.com/pkg.json` — с именем  
- `huesos list` — все репо  
- `huesos use myrepo` — переключаешься  
- `huesos remove myrepo` — удаляешь  
- `huesos -v` — версия  
- `huesos creative` — секретная команда (попробуй сам ☠️)

### Как пользоваться (для дебилов):
```cmd
huesos eblan.browser #халяль браузер
huesos tube.tv #ютубТВ в твоем компе
huesos huesos.builder #халяльный билдер для HUESOS
```
